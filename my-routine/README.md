# 🔥 Build The Best You — Daily Routine App

## Deploy to Vercel (5 minutes)

### Option 1: Drag & Drop (Easiest)
1. Run `npm install` then `npm run build` in this folder
2. Go to **vercel.com** → Sign up free
3. Drag the `build/` folder into Vercel
4. You get a live link instantly ✅

### Option 2: GitHub + Vercel (Best for updates)
1. Push this folder to a GitHub repo
2. Go to **vercel.com** → "New Project" → Import your repo
3. Vercel auto-detects React — just click Deploy
4. Done. Every time you push to GitHub, it auto-deploys ✅

## Run Locally
```bash
npm install
npm start
```
Opens at http://localhost:3000
